# MoneyMate - Expense Tracker

A responsive expense tracker project made with HTML, CSS and JavaScript.

## Features

- Add income
- Add expenses
- Edit transactions
- Delete transactions
- Search transactions
- Filter by type, category and month
- Monthly budget planner
- Budget progress tracking
- Category-wise expense report
- Income vs expense report
- Highest expense and top spending category
- Export transactions as CSV
- Print report
- Dark mode
- localStorage data saving
- Mobile, tablet and laptop responsive design

## How to run

Open `index.html` directly in your browser.

No backend and no installation required.

## How to deploy on Netlify

1. Extract the ZIP file.
2. Go to Netlify.
3. Add new project.
4. Deploy manually.
5. Drag and drop the complete extracted folder.
